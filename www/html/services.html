HTTP Methods

Our server supports the following HTTP methods according to RFC standards

GET

Request data from the server without modifying resources

POST

Submit data to be processed and stored on the server

DELETE

Remove specified resources from the server

GET Request Demo

curl "http://localhost:8080/data?id=42"

POST Request Demo

curl -X POST -F "file=@example.txt" http://localhost:8080/upload

DELETE Request Demo

curl -X DELETE http://localhost:8080/uploads/file.txt

CGI Scripts

index.cgi hello.cgi date.cgi env.cgi
curl http://localhost:8080/cgi-bin/hello.cgi
