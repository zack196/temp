Welcome to WebServ!

This is the default page for the WebServ HTTP server project.

This custom-built server implements:

  • HTTP/1.1 protocol
  • Methods: GET, POST, DELETE
  • Non-blocking I/O
  • Multiple client connections
  • Static file serving
  • Directory listing
  • CGI execution

This server is built as part of the 42 School curriculum requirements.
